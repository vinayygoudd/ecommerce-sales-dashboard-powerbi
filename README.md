##  Project Overview  
This project presents an interactive **E-Commerce Sales Performance Dashboard** built using **Power BI**, providing actionable insights across sales, profitability, customer behaviour, payment preferences, and global market distribution.

The dashboard supports data-driven decision-making by offering real-time KPIs, visual analytics, and filter-driven exploration.

---

##  Key Objectives  
- Analyze **total revenue, profit, orders, and quantity sold**  
- Compare performance across **categories**  
- Identify **monthly revenue patterns** and seasonality  
- Study **profit distribution** across categories  
- Understand **payment method preferences**  
- Visualize global **geographical sales footprint**  
- Provide insights & recommendations for strategic planning  

---

## üóÇ Dataset Summary  
- **Source:** Synthetic e-commerce sales data  
- **Duration:** Multi-year transactional dataset  
- **Rows:** Thousands of sales records  
- **Columns:** 30-dimensional attributes  
- **Key Features:**  
  - Date & time attributes  
  - Product & category information  
  - Pricing, discount, tax, shipping  
  - Customer details  
  - Payment methods  
  - Location data  

Data was cleaned & transformed using **Python (Pandas)** before loading into Power BI.
